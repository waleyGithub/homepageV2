# Writing教程

雅思写作分为两个部分：Task 1 & Task 2. 其中，Task 1占写作总分数的1/3，Task 2占2/3.

写作评分标准：

| Task 1      | Task 2      |
| :----:      | :----:      |
| TA          | TR          |
| CC          | CC          |
| LR          | LR          |
| GRA         | GRA         |
